The 5 Ensemble Models

🔬 Model 1: Evidence Model (25% Weight)

Function: Analyzes legal evidence, precedents, and constitutional references in case opinions.
Training: Learned from 1.69M court opinions to identify precedent-based decision patterns
Accuracy: 92% on test set | Specialization: Precedent & evidence analysis

⚖️ Model 2: Justice Model (20% Weight)

Function: Predicts Supreme Court justice voting patterns based on case characteristics.
Training: Learned voting behaviors from oral arguments and case facts
Accuracy: 88% on test set | Specialization: Justice behavior prediction

🤖 Model 3: ML Model (20% Weight)

Function: General-purpose machine learning classifier for outcome prediction.
Training: Features from case metadata, type, jurisdiction, and temporal factors
Accuracy: 85% on test set | Specialization: Baseline ML classification

📋 Model 4: Amicus Model (20% Weight)

Function: Analyzes amicus curiae (friend-of-the-court) briefs and advocacy patterns.
Training: Learned from briefs and oral arguments to predict advocacy effectiveness
Accuracy: 83% on test set | Specialization: Amicus analysis

🔗 Model 5: Citation Model (15% Weight)

Function: Analyzes citation networks and precedent relationships.
Training: Learned citation patterns and precedent evolution from 1.69M cases
Accuracy: 89% on test set | Specialization: Citation network analysis
