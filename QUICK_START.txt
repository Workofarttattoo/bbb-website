  → Name: bbb-website
  → Make public
  → Create

Step 2: Clone & Setup (3 min)
  → git clone https://github.com/yourusername/bbb-website.git
  → cd bbb-website
  → Copy files:
    - bbb-website-with-streetart.html → index.html
    - BBB-WEBSITE-README.md → README.md
    - bbb-website-gitignore.txt → .gitignore

Step 3: Push to GitHub (2 min)
  → git add .
  → git commit -m "Initial: BBB website with street art design"
  → git push origin main

Step 4: Enable GitHub Pages (1 min)
  → Repository Settings → Pages
  → Source: main branch / root folder
  → Save
  ✅ Live at: https://yourusername.github.io/bbb-website

Step 5: Custom Domain Setup (5-30 min)
  → NameCheap: Add CNAME record (bbb → yourusername.github.io)
  → GitHub: Settings → Pages → Custom domain: bbb.aios.is
  → DNS propagation: 5-30 minutes
  ✅ Live at: https://bbb.aios.is
